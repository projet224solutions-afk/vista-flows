## 📱 **INSTALLATION SUR ANDROID**

### **Méthode 1: Popup Automatique** (Recommandé)

1. **Ouvrir** https://224solution.net sur Chrome Android
2. **Attendre 5 secondes** → Popup apparaît en bas
3. **Cliquer** "Installer maintenant"
4. **Confirmer** l'installation
5. ✅ **Icône ajoutée** à l'écran d'accueil !

### **Méthode 2: Menu Chrome**

1. Ouvrir le site sur Chrome
2. Menu **⋮** (3 points verticaux)
3. **"Installer l'application"** ou **"Ajouter à l'écran d'accueil"**
4. Confirmer
5. ✅ Installé !

### **Méthode 3: Bouton dans l'App**

1. Chercher icône **📥 Télécharger** dans l'interface
2. Cliquer "Installer l'application"
3. Confirmer
4. ✅ Fait !

## 🍎 **INSTALLATION SUR iOS (iPhone/iPad)**

Safari iOS ne supporte pas l'installation automatique, mais vous pouvez ajouter à l'écran d'accueil:

1. **Ouvrir** https://224solution.net sur Safari
2. Bouton **Partage** 📤 (en bas au centre)
3. **"Sur l'écran d'accueil"**
4. Modifier le nom si besoin
5. **Ajouter**
6. ✅ Icône créée !
